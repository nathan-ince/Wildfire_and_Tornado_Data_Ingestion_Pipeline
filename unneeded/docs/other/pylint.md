
---

**`Command`**

```
pylint --version
```

---

**`Command`**

```
pylint file
```

- `file` should be the path of a file

This command will run pylint for a specific file.

---

**`Command`**

```
pylint directory
```

- `directory` should be the path of a directory

This command will run pylint for a whole directory.

**`Example`**

```
pylint src
```
